# ORIGIN Spider Web

Origin(LGNS) 레퍼럴 **조직도 · 자산 추적** 도구. 단일 HTML, 온체인 read-only(서명·자금이동 없음).

라이브: https://spaceechoboy.github.io/origin-spider-web/

## 기능
- **상향/하향 조직도**: 지갑의 추천인 체인·산하 트리를 마인드맵으로 시각화(접기·줌·팬·전체화면)
- **자산 합산**: Polygon + Anubis 양 체인 스테이킹 현재가치(각 체인 시가로 결산)
- **자산 분석**: 동일인(별명) 그룹의 상품별 도넛 + 지갑별 상세
- 별명 · 직급(V0~V6) · 숨김 · 스냅샷 · 색상 설정(전부 브라우저 localStorage, 서버 없음)
- PNG / PDF(A4) 저장 · 모바일 대응(반응형 · 터치 팬/핀치)

## API 키
하향 조직도(하위 발굴)는 무료 [etherscan.io](https://etherscan.io/apidashboard) API 키가 필요합니다.
개인 키를 입력창에 붙여넣으면 이 브라우저에 저장됩니다(읽기 전용 · 자금 접근 불가).
상향·잔액·자산 조회는 키 없이 동작합니다.

## 프라이버시
지갑 별명·직급·숨김·스냅샷은 **이 브라우저에만** 저장되며 어디에도 전송되지 않습니다.
